4. 畸变矫正: UndistortKeyPoints()

实际上,畸变矫正只对单目和RGBD相机输入图像有效,双目相机的畸变矫正参数均为0 ,因为双目相机数据集在发布之前预先做了双目矫正.

双目矫正效果如下,双目矫正将两个相机的成像平面矫正到同一平面上.双目矫正之后两个相机的极线相互平行,极点在无穷远处,这也是我们在
函数ComputeStereoMatches() 中做极线搜索的理论基础.

UndistortKeyPoints() 函数和ComputeImageBounds() 内调用了cv::undistortPoints() 函数对特征点进行畸变矫正

5. 特征点分配: AssignFeaturesToGrid()

在对特征点进行预处理后,将特征点分配到48 行64 列的网格中以加速匹配

成员变量std::vector<std::size_t> mGrid[FRAME_GRID_COLS][FRAME_GRID_ROWS]是一个二维数组,数组中每个元素是对应网格的所有特征点索引列表.
static 成员变量mfGridElementWidthInv 、mfGridElementHeightInv 表示网格宽度/高度,它们在第一次调用Frame 构造函数时被计算
赋值.

函数 vector GetFeaturesInArea(float &x, float &y, float &r, int minLevel, int maxLevel) 获取点 (y,x) 周围半径为 r 的圆域内所有特征点编号.
