视觉转雷达——导航

终于到了导航这一块了。难得啊，废话不多说，下面开始讲解！

注意！注意！注意！

1、在导航时，需要关掉遥控手柄，因为手柄的权限更高，如果不关掉，那么就没有办法启动底盘节点。

2、如果在运行my_car.launch报错，一般会有这个错。电脑开机，一般都要这样，我到底盘launch文件下查看这个错误，就这样的

表示上位机与底盘没有CAN通讯上，需要重新使能，运行下面代码：

sudo ip link set can0 up type can bitrate 500000

1 下面需要启动的launch文件。

启动后就开始导航，先设定目标最大数量为1，然后点击右上方的红色2D Nav Goal，然后就会自动规划出一条线路，点击开始导航。（至于多点导航就没有在这儿演示了）。

总结：其实这个视觉转雷达建图和导航，有很大的问题。在松灵官方演示视频中，他是用的一个VLP16雷达来进行建图，然后没有启动底盘的节点，是从雷达信息中提取里程计odom信息，那个建图效果还不错。但我们刚开始没有用雷达只有相机，所以要想用gmapping建图就需要将视觉转换为雷达，在从雷达信息中提取里程计odom信息，这就造成了很多信息的丢失，所以建图效果不好、导航效果也不好。

所以为了改善这一点，有三种方法。一是启动底盘节点去发布odom（之前松灵机器人的底盘命名都错了，我花了时间找到后，然后改了源码）；二是将相机转雷达中有一个参数叫扫描范围从5改到10；第三就是将livox雷达的雷达装上去。
